Bienvenue sur le script 'mass mailing' de flush

Pour commencer veuillez fournir dans le dossier 'mass mailing' un fichier html nommer 'content.html' qui fournira le contenu du mail envoyer,
vous pouvez ajouter l'attribu {name} dans votre html qui fournira automatiquement le nom de la cible pour chaque mail.
exemple :
<p> salut {name} je t'envoie ce mail grace à 'mass mailing' un script d'envoie de mail en masse incroyable ! </p>

Ensuite veuillez fournir un fichier nommer 'mailing_list.csv' qui permettra au script de savoir les adresses à cibler,
le dossier doit commencer de cette façon 'mail, name' sinon le script ne fonctionnera pas.
exemple :
mail,name
mail@mail.com,mail
jackob@mail.com,jackob
flush@outl.com,flush

après cela veuillez créer un mot de passe d'application pour l'adresse mail qui envoyera les mails. https://support.google.com/accounts/answer/185833?hl=fr

ensuite ouvrez l'invite de commande (CMD)
vous devrez acceder au fichier, pour cela vous devrez écrire "cd {emplacement du fichier}\mass mailling
(pour trouvez l'emplacement du fichier rentrer dans celui ci puis copier l'adresse qui se situera en haut ou il y'aura marquer 'mass mailling'
 puis collez la après avoir écrit cd)
une fois dans le fichier votre interface devrait ressembler à sa: C:\Users\moi\Desktop\mass mailling>
vous n'aura plus qu'a rentrer cette commande et des instructions vous expliquerons comment configurer votre mail: script.py

Merci d'avoir utiliser 'mass mailing'!
